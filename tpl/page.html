Variable example

variable: {$variable}

variable assignment (number=10): {$number=10}
and print (number): {$number}

operation with strings (variable . number): {$variable . $number}

maths operations (number+10): {$number + 10}

a simple modifiers (variable|substr:0,7): {$variable|substr:0,7}

global variable (GLOBALS._SERVER.DOCUMENT_ROOT): {$GLOBALS._SERVER.DOCUMENT_ROOT}

Constants

Constant: {#true#}

Loop example

Simple loop example:
    {loop="week"}
  • {$key} = {$value}
    • {/loop}


Loop example with associative array:
  • ID _ Name _ Color
    • {loop="user"}
  • {$key}) - {$value.name} - {$value.color}
    • {/loop}


Loop an empty array
    {loop="empty_array"}
  • {$key}) - {$value.name} - {$value.color}
    • {else} The array is empty {/loop}

If Example

simple if example:
{if condition="$number==10"}OK! {else}NO!{/if}

example of if, elseif, else example:
{if="substr($variable,0,1)=='A'"}First character is A {elseif="substr($variable,0,1)=='B')First character is B {else}First character of variable is not A neither B {/if}

use of ? : operator (number==10?'OK!':'no')
{$number==10? 'OK!' : 'no'}
